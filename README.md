# Projetos HTML

Este repositório reúne páginas HTML desenvolvidas para treino e aprendizado de desenvolvimento web básico.

## Objetivo
Praticar a estruturação de páginas web utilizando HTML, aplicando conceitos aprendidos em estudos iniciais de front-end.

## Conteúdo
- Páginas HTML simples
- Exercícios de estrutura e semântica
- Projetos de treino

## Tecnologias utilizadas
- HTML5
- CSS3 (quando aplicado)

## Observações
Este repositório é voltado para fins de estudo e prática, acompanhando minha evolução no desenvolvimento web.
